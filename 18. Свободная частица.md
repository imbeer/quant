# –°–≤–æ–±–æ–¥–Ω–∞—è —á–∞—Å—Ç–∏—Ü–∞. 
-- —á–∞—Å—Ç–∏—Ü–∞, –∫–æ—Ç–æ—Ä–∞—è –Ω–∏ —Å –∫–µ–º –∏ –Ω–∏ —Å —á–µ–º –Ω–µ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤—É–µ—Ç(–Ω–∞ –Ω–µ–µ –Ω–µ –¥–µ–π—Å—Ç–≤—É—é—Ç –≤–Ω–µ—à–Ω–∏–µ —Å–∏–ª—ã), –∞ –ø–æ—Ç–æ–º—É –ø–µ—Ä–µ–º–µ—â–∞–µ—Ç—Å—è —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ –∏ –ø—Ä—è–º–æ–ª–∏–Ω–µ–π–Ω–æ => –û–Ω–∞ –æ–±–ª–∞–¥–∞–µ—Ç —Ç–æ–ª—å–∫–æ –∫–∏–Ω–µ—Ç–∏—á–µ—Å–∫–æ–π —ç–Ω–µ—Ä–≥–∏–µ–π. 
–í —Å–≤—è–∑–∏ —Å —ç—Ç–∏–º –æ–ø–µ—Ä–∞—Ç–æ—Ä –ì–∞–º–∏–ª—å—Ç–æ–Ω–∞(= —Å—É–º–º–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –∫–∏–Ω–µ—Ç–∏—á–µ—Å–∫–æ–π –∏ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–π —ç–Ω–µ—Ä–≥–∏–∏) –±—É–¥–µ—Ç –∏–º–µ—Ç—å –¥–ª—è —Ç–∞–∫–æ–π —á–∞—Å—Ç–∏—Ü—ã –≤–∏–¥: $$\hat H = \hat T + \hat U =\frac{mv^2}{2}= \frac{\hat p}{2m} =- \frac{\hbar ^2}{2m}\frac{d^2}{dx^2} = \ –≤ \ \ —Ç—Ä–µ—Ö–º–µ—Ä–Ω–æ–º \ \ –ø—Ä–æ–≤—Ç—Ä–∞–Ω—Å—Ç–≤–µ \ \ = - \frac{\hbar ^2}{2m} \nabla^2$$
# –£—Ä–∞–≤–Ω–µ–Ω–∏–µ –®—Ä–µ–¥–∏–Ω–≥–µ—Ä–∞ –¥–ª—è —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ –¥–≤–∏–∂–µ–Ω–∏—è (—Ç—Ä–µ—Ö–º–µ—Ä–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ). 

$$–£—Ä–∞–≤–Ω–µ–Ω–∏\ \ –®—Ä–µ–¥–∏–Ω–≥–µ—Ä–∞ \ \ i \hbar \frac{d \Psi}{dt} =\hat H \Psi\ \ –≥–¥–µ\ \ \ \Psi (x,y,z,t)=\Psi (\vec ùê´,t);\ \ –∏ \ \ \ U=0 \ \ $$
$$ => \ \ i \hbar \frac{d \Psi (\vec ùê´, t)}{dt} = [-\frac{ \hbar ^2}{2m} \nabla^2 ] \Psi (\vec ùê´, t)$$
# –†–µ—à–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –®—Ä–µ–¥–∏–Ω–≥–µ—Ä–∞ –¥–ª—è —Å–≤–æ–±–æ–¥–Ω–æ–π —á–∞—Å—Ç–∏—Ü—ã.  
1. –ö–æ–≥–¥–∞ –ì–∞–º–∏–ª—å—Ç–æ–Ω–∏–∞–Ω $\hat H$ –Ω–µ –∑–∞–≤–∏—Å–∏—Ç —è–≤–Ω–æ –æ—Ç –≤—Ä–µ–º–µ–Ω–∏, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥ —Ä–∞–∑–¥–µ–ª–µ–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö. –ü—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ–º, —á—Ç–æ –ø–æ–ª–Ω–∞—è –≤–æ–ª–Ω–æ–≤–∞—è —Ñ—É–Ω–∫—Ü–∏—è $\Psi(\vec ùê´, t)$ –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–∞–ø–∏—Å–∞–Ω–∞ –∫–∞–∫ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏–µ –¥–≤—É—Ö —Ñ—É–Ω–∫—Ü–∏–π(–æ—Ç $\vec ùê´$, –∞ –¥—Ä—É–≥–∞—è ‚Äî –æ—Ç t):  $$\Psi(\vec ùê´, t) = \psi(\vec ùê´) f(t)$$
2. –£—Ä–∞–≤–Ω–µ–Ω–∏–µ –¥–ª—è –≤—Ä–µ–º–µ–Ω–Ω–æ–π —á–∞—Å—Ç–∏:$$f(t) = \exp(\frac{-iEt}{\hbar})$$–≥–¥–µ E = const 
3. –£—Ä–∞–≤–Ω–µ–Ω–∏–µ –¥–ª—è –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–æ–π —á–∞—Å—Ç–∏: **–°—Ç–∞—Ü–∏–æ–Ω–∞—Ä–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –®—Ä–µ–¥–∏–Ω–≥–µ—Ä–∞**: $\hat H \psi(\vec ùê´) = E \psi(\vec ùê´)$. $$E \Psi (\vec r) = -\frac{ \hbar ^2}{2m} \nabla^2  \Psi (\vec ùê´)$$
4. –†–∞–∑–¥–µ–ª–∏–º –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –Ω–∞ $-\frac{ \hbar ^2}{2m}$ –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è 1 –ø—Ä–∏ —Å—Ç–∞—Ä—à–µ–π –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–π: $$\nabla^2 \Psi (\vec ùê´)+ \frac{2mE \Psi (\vec r)}{\hbar^2} = 0$$
5. –í–≤–µ–¥–µ–º $k = \sqrt{\frac{2mE}{\hbar^2}}$ - –≤–æ–ª–Ω–æ–≤–æ–µ —á–∏—Å–ª–æ (–ø—Ä–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–π —ç–Ω–µ—Ä–≥–∏–∏ E): $$\nabla^2 \Psi (\vec ùê´)+ k^2\Psi (\vec r) = 0$$
6. –û–±—â–∏–º —Ä–µ—à–µ–Ω–∏–µ–º —ç—Ç–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è —è–≤–ª—è–µ—Ç—Å—è —Å—É–ø–µ—Ä–ø–æ–∑–∏—Ü–∏—è –ø–ª–æ—Å–∫–∏—Ö –≤–æ–ª–Ω –¥–≤–∏–∂—É—â–∏—Ö—Å—è –≤ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–∏:  (–∫—É–± –∏–∑-–∑–∞ —Ç—Ä–µ—Ö–º–µ—Ä–Ω–æ—Å—Ç–∏)
$$\Psi_\vec k (\vec ùê´) = \frac{1}{\sqrt{(2 \pi \hbar )^3}}exp[ i (\vec k \cdot \vec ùê´ )] $$

7. –¢–æ–≥–¥–∞ –∏–∑ (2)
$$\Psi_\vec p (\vec ùê´, t) = \frac {1}{\sqrt{(2 \pi \hbar )^3}} exp[ i (\vec k \cdot \vec ùê´ - \omega t)] $$–≥–¥–µ: $\vec k = \frac {\vec p}{\hbar}; \  \ \ \omega = \frac{E}{\hbar}; E=\frac{p^2}{2m}$ 
# –°–≤—è–∑—å –¥–∞–Ω–Ω–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è —Å –≤–æ–ª–Ω–æ–π –¥–µ-–ë—Ä–æ–π–ª—è
- –ü–æ–ª—É—á–µ–Ω–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –®—Ä–µ–¥–∏–Ω–≥–µ—Ä–∞ –¥–ª—è —Å–≤–æ–±–æ–¥–Ω–æ–π —á–∞—Å—Ç–∏—Ü—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –ø–ª–æ—Å–∫–æ–π –≤–æ–ª–Ω–µ –¥–µ –ë—Ä–æ–π–ª—è: $$œà(ùê´, t) = A exp(\frac{i}{\hbar} (\vec ùê©\cdot \vec ùê´ - Et))$$
- –ï–µ –≤–æ–ª–Ω–æ–≤–æ–π –≤–µ–∫—Ç–æ—Ä $\vec k = \vec p / \hbar$ —Å–≤—è–∑–∞–Ω —Å –∏–º–ø—É–ª—å—Å–æ–º —á–∞—Å—Ç–∏—Ü—ã $\vec p$ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ–º –¥–µ –ë—Ä–æ–π–ª—è $\vec p = \hbar \vec k$.
- –ï–µ —á–∞—Å—Ç–æ—Ç–∞ $\omega = E / \hbar$ (–∏–ª–∏ $\nu = E/h$) —Å–≤—è–∑–∞–Ω–∞ —Å —ç–Ω–µ—Ä–≥–∏–µ–π —á–∞—Å—Ç–∏—Ü—ã E —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ–º –ü–ª–∞–Ω–∫–∞-–≠–π–Ω—à—Ç–µ–π–Ω–∞ $E = \hbar \omega$.
- –ö–≤–∞–¥—Ä–∞—Ç –º–æ–¥—É–ª—è —ç—Ç–æ–π –≤–æ–ª–Ω–æ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ $\vert \Psi \vert^2$ –¥–∞–µ—Ç –ø–ª–æ—Ç–Ω–æ—Å—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è —á–∞—Å—Ç–∏—Ü—ã = `const` => –µ–µ –ø–æ–ª–æ–∂–µ–Ω–∏–µ –ø–æ–ª–Ω–æ—Å—Ç—å—é –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ (—Ä–∞–≤–Ω–æ–≤–µ—Ä–æ—è—Ç–Ω–æ –≤ –ª—é–±–æ–π —Ç–æ—á–∫–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞).
